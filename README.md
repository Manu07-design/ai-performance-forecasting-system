\# AI Performance Forecasting System



A time-series forecasting project that predicts future performance trends using lag-based feature engineering and linear regression.





\# Problem



Given historical weekly performance data, predict next week's marks.





\# Key Concepts Demonstrated



\- Time-series understanding

\- Lag feature engineering

\- Autoregressive forecasting logic

\- Trend visualization

\- Non-shuffled time-based modeling





\# How It Works



1\. Load historical data

2\. Create lag feature (previous week marks)

3\. Train Linear Regression model

4\. Forecast next week

5\. Visualize actual vs predicted trend





\# Tech Stack



\- Python

\- Pandas

\- Matplotlib

\- Scikit-learn



---



\# Learning Outcome



Converted time-series data into supervised learning format using lag features and built a basic autoregressive forecasting engine.
